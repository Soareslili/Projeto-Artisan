# 👜 Artisan Store

O Artisan Store é um e-commerce moderno e elegante desenvolvido para apresentar bolsas de forma sofisticada, proporcionando uma experiência visual agradável e intuitiva para o usuário.

O projeto foi criado com foco em design premium, responsividade e funcionalidades essenciais de uma loja virtual moderna.

---


## 🚀 Tecnologias Utilizadas

- React.js
- Typescript
- Tailwind CSS
- Vite


---

## 🛍️ Sobre o Projeto

O Artisan Store simula uma loja virtual de bolsas, trazendo uma interface moderna, elegante e totalmente responsiva.

A aplicação possui funcionalidades que tornam a experiência do usuário mais dinâmica e interativa, como carrinho de compras, modal de visualização e navegação fluida entre os produtos.

O principal objetivo do projeto foi praticar:

- Componentização no React
- Gerenciamento de estado
- Criação de interfaces modernas
- Responsividade
- Experiência do usuário (UI/UX)
- Estruturação de projetos Front-End

---

## 💻 Funcionalidades

✔️ Catálogo de produtos
